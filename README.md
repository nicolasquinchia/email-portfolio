# Email Development Portfolio

A curated collection of responsive HTML email projects focused on CRM, lifecycle marketing, transactional emails, and promotional campaigns.

This repository showcases production-style email development with an emphasis on responsive layouts, accessibility, cross-client compatibility, and modern CRM/email marketing practices.
